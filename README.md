# Nodo Educativo

Portal web de noticias, herramientas y tecnología educativa para docentes, con foco principal en Chile.

## Estado del proyecto

Proyecto en fase de definición y preparación técnica.

## MVP inicial

- Página principal.
- Noticias Chile.
- Catálogo de herramientas docentes.
- Sección de gadgets educativos.
- Página Quiénes Somos.
- CMS editorial.
- SEO básico.
- Despliegue inicial en Vercel.

## Stack inicial

- Next.js
- TypeScript
- Tailwind CSS
- Sanity CMS
- Vercel
- GitHub

## Desarrollo local

Requisitos:

- WSL.
- Node.js 22 administrado con `nvm` dentro de WSL.
- `node`, `npm` y `npx` deben resolverse desde `~/.nvm/`, no desde rutas de Windows.

Comandos actuales:

- `npm ci` — instalar dependencias desde `package-lock.json`.
- `npm run dev` — iniciar el servidor de desarrollo.
- `npm run lint` — ejecutar la revisión de calidad disponible.
- `npm run build` — generar la compilación de producción.
- `npm start` — iniciar la aplicación compilada después de un build exitoso.
