# Nodo Educativo

Portal web de noticias, herramientas y tecnología educativa para docentes, con foco principal en Chile.

## Estado del proyecto

Proyecto en fase de definición y preparación técnica.

## MVP inicial

- Página principal.
- Noticias Chile.
- Catálogo de herramientas docentes.
- Sección de gadgets educativos.
- Página Quiénes Somos.
- CMS editorial.
- SEO básico.
- Despliegue inicial en Vercel.

## Stack inicial

- Next.js
- TypeScript
- Tailwind CSS
- Sanity CMS
- Vercel
- GitHub

## Stack futuro

- PostgreSQL
- DBeaver
- Newsletter
- Automatizaciones editoriales
