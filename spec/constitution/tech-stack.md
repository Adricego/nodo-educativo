# Stack tecnológico

## Primera versión pública

- Framework: Next.js.
- Lenguaje: TypeScript estricto.
- Estilos: Tailwind CSS.
- Gestión de contenido: Sanity CMS.
- Control de versiones: Git y GitHub.
- Despliegue inicial: Vercel.
- Dominio: por definir; podrá contratarse con un proveedor chileno.

## Base de datos futura

- Motor de base de datos: PostgreSQL.
- Herramienta de administración: DBeaver.
- Uso esperado: newsletter, analítica interna, automatizaciones editoriales,
  favoritos o funcionalidades que requieran datos persistentes.

## Principios técnicos

- Priorizar una arquitectura simple y mantenible.
- Evitar dependencias innecesarias.
- Mantener secretos y claves en variables de entorno.
- No subir archivos `.env` al repositorio.
- Diseñar primero para dispositivos móviles y luego para escritorio.
- Mantener el contenido editorial separado del código.
- Preparar el proyecto para desplegar en Vercel.
